Bases: object

Parameters

  • name (str) – The automaton name.

  • states (Dict[str, BoboState]) – The states of the automaton.

  • transitions (Dict[str, BoboTransition]) – The transitions between states.

  • start_state_name (str) – The name of the start state.

  • final_state_name (str) – The name of the final state.

  • preconditions (List[BoboPredicate]) – Predicates checked before any state’s predicate whereby, if any precondition returns False, the automaton halts.

  • haltconditions (List[BoboPredicate]) – Predicates checked before any state’s predicate whereby, if any haltcondition returns True, the automaton halts.
