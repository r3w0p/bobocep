bobocep.setup.distributed.bobo_dist_manager module¶

class bobocep.setup.distributed.bobo_dist_manager.BoboDistManager(decider: bobocep.decider.bobo_decider.BoboDecider, exchange_name: str, user_id: str, parameters: pika.connection.ConnectionParameters, delay: float, max_sync_attempts: int = 3)¶

Bases: object

A manager for incoming and outgoing data that helps multiple bobocep instances synchronise data between each other using an external message queue system.

Parameters

  • decider (BoboDecider) – The decider to which synchronisation will occur.

  • exchange_name (str) – The exchange name to connect to on the external message queue system.

  • user_id (str) – The user ID to use on the external message queue system.

  • parameters (ConnectionParameters) – Parameters to connect to a message broker.

  • delay (float) – The delay for internal BoboTask threads, in seconds.

  • max_sync_attempts (int, optional) – Maximum attempts to sync with other bobocep instances before giving up, defaults to 3.

cancel()¶

Cancel the manager threads.

start()¶

Run the manager threads.
