class bobocep.rules.states.bobo_state.BoboState(name: str, label: str, predicate: bobocep.rules.predicates.bobo_predicate.BoboPredicate, negated: bool = False, optional: bool = False)¶

Bases: object

A bobocep state.

Parameters

  • name (str) – The state name.

  • label (str) – The state label for grouping multiple states.

  • predicate (BoboPredicate) – The state predicate that, if True, would cause a transition to this state.

  • negated (bool, optional) – Whether the state’s predicate should not happen (i.e. predicate passes evaluation if False), defaults to False.

  • optional (bool, optional) – Whether the state is optional (i.e. will not halt the corresponding automaton if predicate returns False) defaults to False.
