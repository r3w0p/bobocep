bobocep.rules.actions.multi.sequential_action module¶

class bobocep.rules.actions.multi.sequential_action.SequentialAction(actions: List[bobocep.rules.actions.bobo_action.BoboAction], name: str = None, all_success: bool = True, early_stop: bool = True)¶

Bases: bobocep.rules.actions.multi.multi_action.MultiAction

An action that enables the sequential execution of multiple actions within a single BoboAction instance.

Parameters

  • actions (List[BoboAction]) – The actions to execute, in list order.

  • name (str, optional) – The action name, defaults to an empty string.

  • all_success (bool, optional) – All actions must be successful to be a successful execution; otherwise, the success of at least one action would result in an overall success, Defaults to True.

  • early_stop (bool, optional) – Will immediately return False on the first unsuccessful action and not execute any remaining actions, defaults to True. Note that any uncaught exception would still cause an early stop and immediate failure.
