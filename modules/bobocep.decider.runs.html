class bobocep.decider.runs.bobo_run.BoboRun(buffer: bobocep.decider.buffers.shared_versioned_match_buffer.SharedVersionedMatchBuffer, nfa: bobocep.rules.nfas.bobo_nfa.BoboNFA, event: bobocep.rules.events.bobo_event.BoboEvent, start_time: int = None, start_state: bobocep.rules.states.bobo_state.BoboState = None, current_state: bobocep.rules.states.bobo_state.BoboState = None, parent_run: Optional[bobocep.decider.runs.bobo_run.BoboRun] = None, run_id: str = None, version: bobocep.decider.versions.run_version.RunVersion = None, put_event: bool = True, last_process_cloned: bool = False, halted: bool = False)¶

Bases: object

A bobocep antomaton run.

Parameters

  • buffer (SharedVersionedMatchBuffer) – The buffer in which run data will be stored..

  • nfa (BoboNFA) – The automaton with which the run is associated.

  • event (BoboEvent) – The run event.

  • start_time (int, optional) – The time when the run was created. It is used for generating the run ID. Defaults to the event instance’s timestamp.

  • start_state (BoboState, optional) – The start state of the run, defaults to the NFA’s start state.

  • current_state (BoboState, optional) – The current state of the run, defaults to the NFA’s start state.

  • parent_run (BoboRun, optional) – The run which is this run’s parent. It is used when a run has been created as a consequence of non-determinism (i.e. run cloning). Defaults to None.

  • run_id (str, optional) – The ID of the run. Defaults to a run ID consisting of the NFA name and start time.

  • version (RunVersion, optional) – The run version. If a parent run is provided, the version is generated relative to the parent version. Defaults to the run ID.

  • put_event (bool, optional) – Puts the run event into the buffer on instantiation, defaults to True.

  • last_process_cloned (bool, optional) – The last time the run processed an event, a clone occurred, defaults to False.

  • halted (bool, optional) – The run is halted, defaults to False.

Triggers a response when a clone occurs in a run. Newly created runs are considered to be cloned but without a parent run.

Parameters

  • event (BoboEvent) – The event that caused the clone.

  • parent_run_id (str, optional) – The parent run ID, defaults to None.

  • force_parent (bool) – Ensures that a run is found with the parent run ID. Otherwise, an exception is raised.

  • notify (bool) – Whether to notify handler subscribers of the clone.
