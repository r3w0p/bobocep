bobocep.decider.versions.run_version module¶

class bobocep.decider.versions.run_version.RunVersion(parent_version: Optional[bobocep.decider.versions.run_version.RunVersion] = None)¶

Bases: object

The current version of a run, using the identification scheme of the Shared Versioned Match Buffer, as proposed by Agrawal et al. (2008) in their paper “Efficient Pattern Matching over Event Streams”.

Parameters

parent_version (RunVersion) – The parent version. If provided, the new version will copy the parent version’s levels. Defaults to None.

add_level(level: str) → None¶

Adds a new level to the version.

Parameters

level (str) – The level name.

get_previous_version_as_list(decrease_level: int, decrease_incr: int) → List[str]¶

Gets a previous version of the run version as a list of strings.

Parameters

  • decrease_level (int) – How many levels to decrease by. For example, if there are 5 levels, a decrease_level of 2 will go back to level 3.

  • decrease_incr (int) – How many increments of a level to decrease by.

Raises

  • RuntimeError – Decreasing beyond the number of levels.

  • RuntimeError – Decreasing beyond the number of increments in the chosen level.

Returns

The previous version as a list.

get_previous_version_as_str(decrease_level: int, decrease_incr: int, default=None) → str¶

Gets a previous version of the run version as a string, delimited by “.”.

Parameters

  • decrease_level (int) – How many levels to decrease by. For example, if there are 5 levels, a decrease_level of 2 will go back to level 3.

  • decrease_incr (int) – How many increments of a level to decrease by.

  • default (any, optional) – The default value to return if a version string fails to be generated, defaults to None.

Returns

The previous version as a string delimited by “.”. If there are no levels, it returns an empty string.

get_version_as_list() → List[str]¶
Returns

The latest version as a list of strings, where each string is the latest increment of the given level.

get_version_as_str() → str¶
Returns

The latest version as a string, where each level is the latest increment of the given level, separated with the delimeter “.”, for example, “a.b.c”. If there are no levels, it returns an empty string.
