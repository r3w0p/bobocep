bobocep.setup.bobo_complex_event module¶

class bobocep.setup.bobo_complex_event.BoboComplexEvent(name: str, pattern: bobocep.rules.nfas.patterns.bobo_pattern.BoboPattern, action: bobocep.rules.actions.bobo_action.BoboAction = None)¶

Bases: object

A complex event definition. It contains the name of a complex event, the pattern that, if identified, infers the existence of the complex event, and the actions to take if the complex event were to be identified.

Parameters

  • name (str) – The complex event name.

  • pattern (BoboPattern) – The complex event pattern.

  • action (BoboAction, optional) – The action to perform, defaults to no action.

bobocep.setup.bobo_setup module¶

class bobocep.setup.bobo_setup.BoboSetup(delay: float = 0.5, recursive: bool = True, max_queue_size: int = 0, max_recent: int = 1)¶

Bases: bobocep.setup.distributed.outgoing.dist_outgoing_subscriber.IDistOutgoingSubscriber

A class to set up a working bobocep process.

Parameters

  • delay (float, optional) – The delay, in seconds, for internal threads to wait before repeating its processing task, defaults to 0.5.

  • recursive (bool, optional) – Whether CompositeEvent instances should be put back into the system on generation, defaults to True.

  • max_queue_size (int, optional) – The maximum data queue size, defaults to 0 (infinite).

  • max_recent (int, optional) – The maximum number of recent CompositeEvent instances generated by this handler to pass to runs. Minimum of 1, defaults to 1.
