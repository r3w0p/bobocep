Bases: bobocep.rules.events.bobo_event.BoboEvent

An action event.

Parameters

  • timestamp (int) – The event timestamp indicating when it was first generated.

  • name (str) – The action name.

  • success (bool) – True if action was successfully executed, False otherwise.

  • for_event (BoboEvent) – The event for which the action was triggered.

  • exception (str, optional) – The name of an exception, if one was raised during action execution. Defaults to an empty string.

  • description (str, optional) – The description provided with an exception, if one was raised. Defaults to an empty string.

  • data (Dict[str, str], optional) – The event data, defaults to an empty dict.

  • event_id (str, optional) – The event ID, defaults to a randomly generated ID.
