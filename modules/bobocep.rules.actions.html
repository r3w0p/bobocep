bobocep.rules.actions.rate_limit_action module¶

class bobocep.rules.actions.rate_limit_action.RateLimitAction(name: str = None, limit_dict: Dict[str, int] = None, rate_other: int = 0)¶

Bases: bobocep.rules.actions.bobo_action.BoboAction

Ensures that CompositeEvent instances of a given name are only produced at a given rate i.e. one event with name n in t seconds. If multiple instances are created within the rate, the subsequence ones are dropped i.e. the action returns False.

Parameters

  • name (str, optional) – The action name, defaults to an empty string.

  • limit_dict (Dict[str, int], optional) – Key-value pairs, where the key is the CompositeEvent name and the value is the desired rate, in seconds. Defaults to an empty dict.

  • rate_other (int, optional) – The rate for all other events that have not been named, defaults to 0.

get_limits() → Dict[str, int]¶
Returns

A dict, where the keys are the names to limit and the values are the desired rates for each name, in seconds.
